Simplifies working with nested objects (and arrays).

This object is intended for internal formFor use only and is not exposed to the $injector.

Instance Methods

flattenAttribute(fieldName)

Converts a field name (which may contain dots or array indices) into a string that can be used to key an object (e.g. items[0].name would be converted into items___0______name).

flattenObjectKeys(object)

Crawls an object and returns a flattened set of all attributes using dot notation. This converts an Object like: {foo: {bar: true}, baz: true} into an Array like ['foo', 'foo.bar', 'baz'].

readAttribute(object, fieldName)

Returns the value defined by the specified attribute. This function guards against dot notation for nested references (e.g. foo.bar).

writeAttribute(object, fieldName, value)

Writes the specified value to the specified attribute. This function guards against dot notation for nested references (e.g. foo.bar).
