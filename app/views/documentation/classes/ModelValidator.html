Used by formFor to determine if each field in the form-data passes validation rules.

This service can be accessed via $injector.get('ModelValidator') although it is not intended for use outside of the formFor module/library.

Static Methods

isCollectionRequired(fieldName, validationRuleSet)

Determines if the specified collection is required (requires a minimum number of items).

isFieldRequired(fieldName, validationRuleSet)

Determines if the specified field is flagged as required.

validateAll(formData, validationRuleSet)

Validates the model against all rules in the validationRules. This method returns a promise to be resolved on successful validation or rejected with a map of field-name to error-message.

validateCollection(formData, fieldName, validationRuleSet)

Validate the properties of a collection (but not the items within the collection). This method returns a promise to be resolved on successful validation or rejected with an error message.

validateField(formData, fieldName, validationRuleSet)

Validates a value against the related rule-set (within validationRules). This method returns a promise to be resolved on successful validation. If validation fails the promise will be rejected with an error message.

validateFields(formData, fieldNames, validationRuleSet)

Validates a value against the related rule-set (within validationRules). This method returns a promise to be resolved on successful validation. If validation fails the promise will be rejected with an error message.
