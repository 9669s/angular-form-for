Shared helper methods used by formFor field directives.

This service can be accessed via $injector.get('FieldHelper') although it is not intended for use outside of the formFor module/library.

Static Methods

manageLabel($scope, $attributes, humanizeValueAttribute)

Determines the field's label based on its current attributes and the FormForConfiguration configuration settings. Also watches for changes in the (attributes) label and updates $scope accordingly.

manageFieldRegistration($scope, $attributes, formForController)

Registers a form field and stores the bindable object returned on the $scope. This method also unregisters the field on $scope $destroy.
