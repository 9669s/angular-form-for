IE8 Support

The formFor library officially supports IE9+, but IE8 support can be achieved with a few simple steps.

Do not change input type

According to Microsoft:

The type property is read/write-once, but only when an input element is created with the createElement method and before it is added to the document.

Polyfills

Function.prototype.bind was added in ECMA 5.1 and so is only available on IE9+. Mozilla provides a simple polyfill to add support though: https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Function/bind

Object.keys was added in ECMA 5.1 and so is only available on IE9+. Mozilla provides a polyfill for it though: https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object/keys
